# Travel Guide Generator / 旅游攻略生成器

本项目包含一个旅游攻略生成器的 AI agent 指令，跨平台通用。

当用户要求生成某个目的地（城市/地区/线路）的旅游攻略时，请读取 `AGENT_GUIDE.md` 并按其流程执行：

1. 联网搜索（优先小红书热门推荐）目的地的景点/交通/住宿/美食/避坑等实时信息
2. 按标准结构整理生成攻略正文（Markdown，含图片占位符）
3. 输出文件到当前目录（默认只生成攻略正文；售卖模式额外生成上架文案和差异化补充清单）

详细流程、参数、写作规范见 `AGENT_GUIDE.md`，参考资料在 `references/` 目录。
